# View Component

VIewComponent 類似 ViewPartial，參數直接來自檢視元件的引動過程，而不是來自模型繫結。

ViewComponent無法直接處理Request (處理Request仍然需要依靠Controller)

ViewComponent針對更新的元件，需要重新編譯後才能在畫面更改
